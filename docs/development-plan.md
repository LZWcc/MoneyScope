# MoneyScope 开发计划

## Stage 1：项目初始化 ✅

1. 初始化 Git 仓库。
2. 创建项目目录结构。
3. 编写 README、CLAUDE、AI_LOG 模板。
4. 编写需求分析和数据库设计文档。
5. 实现 SQLite 数据库初始化。

## Stage 2：核心数据功能 ✅

1. 定义 Transaction 和 Budget 数据类。
2. 实现交易记录新增、删除、查询。
3. 实现月度统计和分类统计。
4. 准备示例 CSV 数据和演示数据库。
5. 编写 pytest 自动测试（22 个用例全部通过）。

1. 实现记录录入页面（含快捷记账按钮）。
2. 实现交易列表页面（含下拉选择删除）。
3. 实现筛选控件（月份、类型、分类三维度）。
4. 实现月度汇总展示（指标卡片）。

## Stage 4：图表和预算 ✅

1. 实现分类消费饼图。
2. 实现月度每日趋势图（自适应布局）。
3. 实现预算设置、预算列表展示和删除。
4. 实现预算超额/接近提醒。

## Stage 5：导入导出与分类管理 ✅

1. 实现 CSV 导入（逐行校验，错误行报告）。
2. 实现 CSV 导出（UTF-8-sig 编码）。
3. 实现分类管理（添加/删除收支分类）。

## Stage 6：代码重构 ✅

1. 拆分 main.py 为独立模块（constants、styles、components、_pages/）。
2. 每个文件不超过 170 行，职责清晰。
