# MoneyScope 开发计划

## Stage 1：项目初始化

1. 初始化 Git 仓库。
2. 创建项目目录结构。
3. 编写 README、CLAUDE、AI_LOG 模板。
4. 编写需求分析和数据库设计文档。
5. 实现 SQLite 数据库初始化。

## Stage 2：核心数据功能

1. 定义 Transaction 和 Budget 数据类。
2. 实现交易记录新增、删除、查询。
3. 实现月度统计和分类统计。
4. 准备示例 CSV 数据。
5. 编写基础测试或手动测试记录。

## Stage 3：Streamlit 界面

1. 实现记录录入页面。
2. 实现交易列表页面。
3. 实现筛选控件。
4. 实现月度汇总展示。

## Stage 4：图表和预算

1. 实现分类消费图表。
2. 实现月度趋势图。
3. 实现预算设置和提醒。

## Stage 5：导入导出

1. 实现 CSV 导入。
2. 实现 CSV 导出。
3. 处理错误 CSV 文件。
4. 在测试报告中记录结果。

## Stage 6：课程提交材料

1. 更新 README。
2. 完善 AI_LOG。
3. 补充测试报告。
4. 撰写项目报告。
5. 准备 PPT 和演示视频脚本。
